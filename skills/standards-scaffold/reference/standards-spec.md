# Standards（智能路由）

## Overview

项目统一开发规范。自动检测工作目录类型，加载对应规范。

## 默认加载

所有项目默认加载以下规范：

- **架构模式**：`reference/backend/patterns/`
- **测试规范**：`reference/backend/testing/unittest.md`
