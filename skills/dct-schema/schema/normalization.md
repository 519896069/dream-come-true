# 阶段一：需求标准化

## 阶段产物

| 产物 | 说明 |
|------|------|
| `prd/{date}-{需求}/checkpoint.md` | 验收标准 |
| `prd/{date}-{需求}/user-stories.md` | 用户故事 |

## 产物规范

### 验收标准 `prd/{date}-{需求}/checkpoint.md`

```markdown
# 验收点清单

## 验收点

1. 验收点1
2. 验收点2
...

```

## 用户故事 `prd/{date}-{需求}/user-stories.md`

用以下格式编写用户故事：
- 用户故事使用「作为...我希望...以便...」格式
- 每个用户故事包含：标题、角色、行为、价值

```markdown
## 用户故事

### US-001
**作为** [角色]
**我希望** [行为]
**以便** [价值]

### US-002
...
```

## 质量标准

- 验收点覆盖所有功能需求
- 用户故事完整且无遗漏
- 需求可测试、可验证
- 用户故事拆分，保证原子性，且各故事间没有交集
