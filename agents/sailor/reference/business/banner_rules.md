# 轮播图管理业务规则

## 一、业务功能点

| 功能编号 | 功能名称 | 说明 |
|---------|---------|------|
| F001 | 轮播图列表 | 分页展示所有轮播图，支持查看ID、发布人、位置、预览、目标地址、添加时间 |
| F002 | 轮播图创建 | 上传图片，填写标题、跳转地址、位置、上下线状态、置顶状态 |
| F003 | 轮播图编辑 | 修改已有轮播图的各项属性 |
| F004 | 轮播图删除 | 永久删除轮播图 |
| F005 | 轮播图位置管理 | 支持三种位置：首页(空)、百场万企(BCWQ)、发榜揭榜(JBGS) |
| F006 | 上下线管理 | 控制轮播图是否在前台展示 |
| F007 | 置顶管理 | 置顶的轮播图在同位置下优先展示 |

## 三、数据库设计

### 3.1 ER 图

```
┌─────────────────────────────────────────────┐
│              banner 表 (单表)                 │
├─────────────────────────────────────────────┤
│ id (PK, bigint)                             │
│ user_id (bigint) ──────────┐                │
│ title (varchar)            │                │
│ url (text)                 │ 无外键关联      │
│ target_type (tinyint)       │ 纯业务表        │
│ target_url (text)          │                │
│ status (tinyint)           │                │
│ top (tinyint)             │                │
│ position (varchar)         │                │
│ created_at (datetime)      │                │
│ updated_at (datetime)     │                │
└─────────────────────────────┘                │
```

### 3.2 表结构

| 字段名 | 类型 | 注释 | 说明 |
|--------|------|------|------|
| id | bigint | banner_id | 主键 |
| user_id | bigint | 创建人id | 默认0 |
| title | varchar(50) | 图片标题 | 轮播图显示标题 |
| url | text | 图片地址 | 存储在 /public/upload/YYYYMMDD/ 下 |
| target_type | tinyint(1) | 目标地址类型 | 默认0 |
| target_url | text | 目标地址 | 点击轮播图跳转的URL |
| status | tinyint(1) | 上下线状态 | 0=下线, 1=上线 |
| top | tinyint(1) | 置顶状态 | 0=默认, 1=置顶 |
| position | varchar(32) | banner位置 | ''=首页, 'BCWQ'=百场万企, 'JBGS'=发榜揭榜 |
| created_at | datetime | 创建时间 | |
| updated_at | datetime | 更新时间 | |

**索引**：
- PRIMARY KEY (`id`)
- INDEX `idx_created_at_status` (`created_at`, `status`)

## 四、API 文档

### 4.1 获取轮播图列表

- **URL:** `/api/banner`
- **Method:** `GET`
- **Request:**
  | 参数名 | 类型 | 必填 | 说明 |
  |--------|------|------|------|
  | page | int | 否 | 页码，默认1 |
  | position | string | 否 | 位置筛选：空字符串=首页, 'BCWQ'=百场万企, 'JBGS'=发榜揭榜 |

### 4.2 获取轮播图详情

- **URL:** `/api/banner/{id}`
- **Method:** `GET`

### 4.3 创建轮播图

- **URL:** `/api/banner`
- **Method:** `POST`
- **Request:**
  | 参数名 | 类型 | 必填 | 说明 |
  |--------|------|------|------|
  | title | string | 是 | 轮播图标题 |
  | url | string | 是 | 图片上传后的路径 |
  | target_url | string | 是 | 点击跳转地址 |
  | status | int | 是 | 0=下线, 1=上线 |
  | top | int | 是 | 0=默认, 1=置顶 |
  | position | string | 否 | 位置：''=首页, 'BCWQ'=百场万企, 'JBGS'=发榜揭榜 |

### 4.4 更新轮播图

- **URL:** `/api/banner/{id}`
- **Method:** `PATCH`

### 4.5 删除轮播图

- **URL:** `/api/banner/{id}`
- **Method:** `DELETE`

## 五、业务逻辑说明

### 5.1 图片上传流程

1. 前端通过 el-upload 组件上传图片到 `/api/upload`
2. 上传成功后返回临时路径
3. 创建/更新轮播图时，BannerBusiness 将图片从临时目录移动到 `/public/upload/YYYYMMDD/` 目录
4. 数据库存储移动后的相对路径

### 5.2 排序规则

- 首先按 `top` 字段倒序（置顶优先）
- 其次按 `created_at` 倒序（新建优先）
- 前台展示时限制返回5条记录

### 5.3 位置说明

| position值 | 位置名称 | 前台展示位置 |
|------------|---------|-------------|
| '' (空) | 首页 | 首页轮播区 |
| 'BCWQ' | 百场万企 | 百场万企活动页轮播区 |
| 'JBGS' | 发榜揭榜 | 发榜揭榜页轮播区 |
